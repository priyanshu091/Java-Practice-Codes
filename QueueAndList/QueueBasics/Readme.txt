### Queue Operations:
- offer() - Safe adding
- poll() - Safe removal with null return
- peek() - View without removal
- element() - View with exception
- remove() - Remove with exception
- isEmpty() - Empty check
- size() - Count elements

### List Operations:
- add() - Adding elements
- get(index) - Access by position
- remove(index) - Remove by position
- indexOf(element) - Find position
- contains(element) - Check existence
- size() - Count elements
- isEmpty() - Empty check

- Custom ordering (Comparator)
- Lambda expressions
- Integer.compare() usage

- Constructors
- Getters/Setters
- toString() override
- Custom class design
