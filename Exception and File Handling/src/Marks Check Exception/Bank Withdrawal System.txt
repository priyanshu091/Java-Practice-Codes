# Marks Validation Program (Java)

## ?? About This Program
This program validates student marks using Java Exception Handling.
It checks whether entered marks are within valid range.

## ?? What this program does
- Takes marks input from user
- Validates marks using a separate method
- Throws exception if marks are invalid
- Handles exception in main method

## ?? Validation Rules
- Marks must be between 0 and 100
- Marks less than 0 or greater than 100 are considered invalid

## ?? Custom Exception Used
InvalidMarksException  
- Thrown when marks are outside valid range

## ?? Java Concepts Used
- Custom Exception class
- throw keyword
- throws keyword
- try-catch-finally
- Method-based validation

## ? Output Behavior
- Valid marks are printed
- Invalid marks show error message
- Finally block prints completion message

## ?? Purpose
This program is designed to practice:
- Custom exceptions
- Proper exception flow
- Clean and structured Java code
